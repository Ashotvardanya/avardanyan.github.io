Ծանոթացում լեզվին.

Մեր դասընթացները կսկսենք JavaScript լեզվի ուսումնասիրումից : Դա ծրագրավորման լեզու է, որից սկսում են շատ ծրագրավորողներ: Բայց ինչո՞ւ հենց JavaScript :

1.Այն դյուրին է:

Այն շատ հեշտ է սկսնակների համար և շատ լեզուների հետ ընդհանրություններ ունի:

2.Համապատասխանությունը բրաուզերների հետ :

Այդ լեզվով գրված ծրագրերը համապատասխանում են գրեթե բոլոր բրաուզերներին և բարդություններ չեն առաջացնի:

3. Հետաքրքրությունը լեզվի հանդեպ:

2015թ. ից մարդկանց հետաքրքրությունը այս լեզվի նկատմամբ աճել է 25%-ով :

Այս դասընթացը սկսելու համար մեզ պետք է ներբեռնել ինչ-որ ծրագիր, որ կօգնի մեզ ծրագրավորման գործում (կտարբերակի մեր ներմուծածը, կներմուծի արդեն իր մեջ գոյություն ունեցող ֆունկցիաների պատրաստի տարբերակները և այլն) ,քանի որ Windows օպերացիոն համակարգում արդեն իսկ եղած Notepad և WordPad ծրագրերով ընդհանրապես հարմար չէ ծրագրավորել:Շատ լավ ծրագիր է հանդիսանում Sublime Text 3-ը :Ահա թե ինչպես ներբեռնել այդ ծրագիրը.
